## INTMAX とは

INTMAX は、ゼロ知識証明（ZKP）ベースのレイヤー2（Layer 2）プロトコルです。ステートレスアーキテクチャ（Stateless Architecture）を採用し、スケーラブルかつプライバシーを保護した決済を実現します。
