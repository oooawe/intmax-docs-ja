# インストール

`INTMAX WalletSDK` を使い始めるには、npm、yarn、pnpm などのパッケージマネージャーでインストールします：

{% tabs %}
  {% tab title="npm" %}
    ```sh npm install intmax-walletsdk ```
  {% endtab %}
  {% tab title="yarn" %}
    ```sh yarn add intmax-walletsdk ```
  {% endtab %}
  {% tab title="pnpm" %}
    ```sh pnpm add intmax-walletsdk ```
  {% endtab %}
{% endtabs %}
