# エラーコード

エラーコードは EIP-1193 を継承し、以下のエラーコードが追加されています。

| ステータスコード | 名称                | 説明                                              |
| ----------- | ------------------- | ------------------------------------------------- |
| 5001        | Chain not Supported | Namespace で指定されたチェーン ID をウォレットがサポートしていない |
| 5100        | Popups blocked      | 何らかの理由でウォレットを開けなかった                      |
